# Computation function for rebuild mc-ind weekly from daily data.

Computation function for rebuild mc-ind weekly from daily data.

## Usage

``` r
.dailyToWeekly(dailydata, opts = simOptions(), type)
```

## Arguments

- dailydata:

  antaresData for daily timestep

- opts:

  study opts

- type:

  type of data (areas, links, clusters, clustersRes)
