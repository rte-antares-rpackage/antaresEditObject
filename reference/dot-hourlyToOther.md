# Computation function for rebuild mc-ind daily, monthly and annual from hourly data.

Computation function for rebuild mc-ind daily, monthly and annual from
hourly data.

## Usage

``` r
.hourlyToOther(hourlydata, timeStep, type)
```

## Arguments

- hourlydata:

  antaresData for hourly timestep

- timeStep:

  timestep of aggregation (daily, monthly and annual, NO weekly)

- type:

  type of data (areas, links, clusters, clustersRes)
