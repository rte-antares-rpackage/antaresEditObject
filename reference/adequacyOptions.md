# Adequacy patch parameters for creating an area

Adequacy patch parameters for creating an area

## Usage

``` r
adequacyOptions(adequacy_patch_mode = "outside")
```

## Arguments

- adequacy_patch_mode:

  character, default to "outside"

## Value

a named list

## Examples

``` r
adequacyOptions()
#> $`adequacy-patch-mode`
#> [1] "outside"
#> 
```
