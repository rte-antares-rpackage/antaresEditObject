# Correspondence between arguments of `updateOutputSettings` and actual Antares parameters.

Correspondence between arguments of `updateOutputSettings` and actual
Antares parameters.

## Usage

``` r
dicoOutputSettings(arg)
```

## Arguments

- arg:

  An argument from function `updateOutputSettings`.

## Value

The corresponding Antares general parameter.

## Examples

``` r
dicoOutputSettings("result.format") # "result-format"
#> [1] "result-format"
```
