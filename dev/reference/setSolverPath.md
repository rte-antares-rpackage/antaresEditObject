# Set path to Antares Solver

Set path to Antares Solver

## Usage

``` r
setSolverPath(path)
```

## Arguments

- path:

  (optional) Path to the solver (e.g. `antares-6.0-solver.exe` in \bin
  directory where Antares is installed). If missing, a window opens and
  lets the user choose the directory of the simulation interactively.

## Examples

``` r
if (FALSE) { # \dontrun{

setSolverPath(path = "C:/antares/bin/antares-6.0-solver.exe")

} # }
```
